# Solana Get Balance

This project demonstrates the usage of Solana Client and Solana Program in Rust to interact with the Solana blockchain and get balance from public key.

## Prerequisites

Before getting started, make sure you have the following prerequisites:

- Rust: Install Rust on your machine using [Rustup](https://rustup.rs/).
- Solana CLI: Install the Solana Command-Line Tools.


## Installation

1. Clone the repository:
2. Navigate to the project directory:
3. Build using `cargo build` command
4. Run using `cargo run` command
